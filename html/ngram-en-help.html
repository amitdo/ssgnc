Google N-gram Search HowTo

Query

Please input white-space separated unigrams.
Ex: "apple juice", "is a"

Three kinds of matching methods are available.

  • "Unordered": Searches n-grams which contain given unigrams in any order.
    Users can specify the minimum number of occurrences by passing the same unigram more than once.
    Ex: Query "day day" matches "day by day" but not "of the day."
  • "Ordered": Searches n-grams which contain given unigrams in order.
    Ex: Query "milk coffee" matches "milk and coffee" but not "coffee and milk."
  • "Phrase": Searches n-grams which contain a given phrase.
    Ex: Query "milk coffee" matches "frothy milk coffee" but not "milk and coffee."
  • "Fixed": Searches n-grams using a wildcard character '*'.
    Ex: Query "milk *" matches "milk coffee" and "milk chocolate."

N-grams

Please input 'k' of target k-grams.
Ex: If "3" is given, it searches 3-grams.
Ex: If "1-7" is given, it searches 1-grams to 5-grams.
Ex: If "2,4-5" is given, it searches 2-grams, 4-grams and 5-grams.

English Google N-gram Resources consist of 1-grams to 5-grams.
Japanese Google N-gram Resources consist of 1-grams to 7-grams.

No. Results

Please input the maximum number of search results.
Ex: If "1" is given, it returns only one n-gram which has the maximum frequency.
Ex: If "100" is given, it returns the top 100 n-grams.
Ex: If "all" is given, it returns all n-grams matching a given query.

If the number of search results will be large (more than 10,000), please choose "Download." Otherwise, a browser might freeze, in particular Internet Explorers.

Min. Frequency

Please input the minimum frequency of search results.
Ex: If "10" is given, it returns all n-grams matching a given query.
Ex: If "1000" is given, it returns n-grams with frequencies not less than 1000.

English Google N-gram Resources consist of unigrams with frequencies not less than 200 and n-grams with frequencies not less than 40.
Japanese Google N-gram Resources consist of unigrams with frequencies not less than 50 and n-grams with frequencies not less than 20.

Results

Please choose how to get search results.
"Browse": Displays search results on a browser.
"Save": Popups a download dialog for search results.

If the number of search results will be large (more than 10,000), please choose "Download." Otherwise, a browser might freeze, in particular Internet Explorers.
